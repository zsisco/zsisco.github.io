Zachary Sisco is a Masters student at Wright State University. He is currently working with Dr. Adam Bryant in the ACSL (Autonomous Cyber Systems Lab).

Research

Zach is interested in mathematical models and representations for software reverse engineering problems.

Current Research Project

Zach is currently researching using formal methods to model the concept assignment problem for assembly language representations of programs.
The concept assignment problem (also referred to as concept location or feature location) is the problem of mapping concepts to locations in a program.
This is difficult in assembly because assembly instructions lack semantic information and higher-level programming constructs compared to source code representations of programs. Automating concept assignment improves a reverse engineer's comprehension of executable software by determining what concepts and features exist in a program.
